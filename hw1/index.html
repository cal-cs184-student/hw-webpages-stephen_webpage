Task 1: Drawing Single-Color Triangles

My implementation starts by checking the winding order of the triangle, ensuring it is oriented with points labeled in a counter-clockwise (CCW) order. This is done by checking the sign of the cross product of two edges of the triangle.

I then compute the bounding box of the triangle by finding the minimum and maximum X and Y values. The bounding box ensures the triangle is contained within it by flooring the minimum values and ceiling the maximum values.

Each pixel's center within the bounding box is checked to determine if it is inside the triangle. If it is, the fill_pixel function is called.
